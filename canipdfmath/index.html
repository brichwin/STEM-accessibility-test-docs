PDF/UA-2 Math Encoding Test Results

Test results for PDF/UA-2 documents containing math encoded in combinations of three formats: Associated Files (AF), Structure Elements (SE), and Alt text on the formula tag. Each combination was tested in Adobe Acrobat Pro and Firefox with NVDA and JAWS screen readers.

Encoding formats tested:

  • AF: MathML stored as an Associated File attached to the formula object
  • SE: MathML embedded directly in the PDF tag tree as Structure Elements
  • Alt text: Auto-generated plain text alternative on the formula tag (Alt attribute) using \tagpdfsetup{math/alt/use}
